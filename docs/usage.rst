表端错误提示
~~~~~~~~~~~~~~~

本系统中如果在卡操作过程出现错误,液晶会提示错误,LED 闪烁报警。

错误如表所示

.. list-table::

	* - 序号
	  - 错误显示
	  - 错误原因
	* - 1
	  - Error  01
	  - 校验和出错。
	* - 2
	  - Error  03
	  - 二级低压，不能进行卡操作	
	* - 3
	  - Error  04
	  - 未开户或未刷测试卡	
	* - 4
	  - Error  05
	  - 用户卡表号不对应	
	* - 5
	  - Error  06
	  - 提前拔卡	
	* - 6
	  - Error  07
      - 超出最大购买量	
	* - 7
	  - Error 12
	  - 刷卡过快或者不是本客户系统卡	
	* - 8
	  - Error 15
	  - 卡片确认失败，线圈未连接好或者有磁场干扰	
	* - 9
	  - Error 16
	  - 阶梯价设置错误，不能为 0	
	* - 10
	  - Error 17
	  - 换表卡二次换出时表号不对应	
	* - 11
	  - Error 18
	  - 充值不成功，表端充值次数大于卡次数
